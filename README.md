This is a beginner‑friendly e‑commerce web application built with PHP and MySQL.
It demonstrates the basic flow of an online shop:

🛍️ Product Catalog – Displays products from the database with name, description, price, and image.

🛒 Shopping Cart – Users can add products to their cart, update quantities, and view totals.

👤 User Authentication – Registration, login, and logout system using sessions.

📑 Orders – Checkout process that saves orders linked to logged‑in users.

📜 Order History – Users can view their past orders and their status (Pending, Shipped, Completed).

🔧 Admin Features – Shop owner can view all orders and update their status.
